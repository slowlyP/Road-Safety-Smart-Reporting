```
📌 프로젝트 개요

AI 기반으로 도로 위 낙하물을 탐지하고,
사용자에게 위험 정보를 제공하는 웹 서비스.

🧩 전체 시스템 구조
사용자 → (Frontend) → API → Service → Repository → DB
                              ↓
                           AI 모델

```

```
👤 사용자 기능 (User)

1. 회원 기능
회원가입
로그인 / 로그아웃
회원정보 수정
회원 탈퇴

2. 신고 기능
이미지 / 영상 업로드
낙하물 탐지 (AI 분석 자동 실행)
신고 등록

3. 내 신고 관리
내 신고 목록 조회
신고 상세 조회
신고 수정

4. 탐지 현황 (핵심 기능 🔥)
최근 낙하물 리스트 조회
위험도 표시 (주의 / 경고 / 긴급)
지도 기반 위치 표시
탐지 객체 (box, rock, tire 등)
신뢰도(confidence) 표시

5. 경로 위험 확인
출발지 / 도착지 입력
경로 내 위험 구간 탐색
낙하물 위치 기반 위험 안내

```

```
🛠 관리자 기능 (Admin)

1. 대시보드
전체 신고 수
위험도 통계
최근 탐지 현황

2. 신고 관리
전체 신고 목록 조회
신고 상세 확인
상태 변경 (처리중 / 완료 등)

3. 사용자 관리
회원 목록 조회
권한 변경 (user / admin)

4. AI 분석 관리
탐지 결과 확인
모델 비교 결과 조회

5. 실시간 알림 (🔥 핵심 차별점)
새로운 낙하물 발생 시 즉시 알림
WebSocket 기반 실시간 전송
위험도 높은 경우 강조 알림
🤖 AI 기능
객체 탐지 모델 (YOLO 기반)
탐지 대상:
box
rock
tire
confidence 기반 필터링
bbox 좌표 저장

```

```

   ↓
```

```
🚀 핵심 포인트 (이 프로젝트의 강점)
단순 CRUD가 아닌 AI + 실시간 시스템 결합
WebSocket 기반 실시간 알림 구현
서비스 레이어 기반 구조 설계 (확장성 고려)

데이터 증가 기반 재학습

📎 기타 참고
API는 REST 기반으로 설계됨
서비스 계층 중심 구조로 유지보수 용이
```
