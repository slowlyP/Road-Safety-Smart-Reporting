# 🛣️ AI 기반 도로 사고 스마트 신고 시스템 (Backend)

> **YOLO 객체 탐지와 LLM(Gemini)을 결합하여 도로 위 위험 요소를 자동으로 분석하고 신고하는 스마트 관제 시스템입니다.**

---

## 🚀 Key Features (본인 담당 핵심 기능)

### 1. AI 텍스트 생성 파이프라인 (Gemini 1.5 Flash)
- **자동화:** 분석된 객체(낙하물, 사고 등) 정보를 바탕으로 신고 제목과 내용을 AI가 자동 생성하여 신고 편의성 극대화.
- **안정성:** API 할당량 초과나 네트워크 오류 시 시스템이 중단되지 않도록 **Fallback 로직**을 설계하여 서비스 연속성 확보.
- **UX 최적화:** AI의 영문 라벨을 사용자 친화적인 한국어 데이터로 매핑하여 직관적인 정보 제공.

### 2. 스마트 데이터 전처리 및 위치 매핑
- **Exif 분석:** 업로드된 이미지의 메타데이터에서 위경도 좌표를 추출하여 사고 발생 위치 자동 매핑 기능 구현.
- **리소스 최적화:** 대용량 파일(50MB) 제한 및 미디어 확장자 필터링 로직을 통해 서버 자원 및 보안 관리.

### 3. 실시간 관리자 알림 시스템
- **데이터 정합성:** 데이터베이스 트랜잭션 성공 시에만 실시간 알림을 전송하도록 설계하여 데이터 무결성 보장.

---

## 🛠️ Technical Decision (기술적 의사결정)
- **Layered Architecture:** `Route - Service - AI Service`로 계층을 분리하여 코드의 재사용성과 유지보수성 향상.
- **Error Handling:** 외부 API 의존성을 관리하기 위해 예외 처리를 강화하고, 에러 발생 시 DB 롤백 및 물리 파일 삭제 연동으로 시스템 청결도 유지.

---

## 📦 Tech Stack
- **Language:** Python 3.11
- **Framework:** Flask
- **AI:** Google Gemini API, YOLOv8, RT-DETR
- **Database:** MySQL
- **Tool:** Git, GitHub
